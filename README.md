
# docxjs
Javascript port of [python-docx](https://github.com/python-openxml/python-docx).  

## Installation
```shell
npm i @zhangyu836/docxjs
```

## Documentation
Please read [python-docx documentation](https://python-docx.readthedocs.org/en/latest/) for now.
